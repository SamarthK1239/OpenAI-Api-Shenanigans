### "API key not found in environment variables!"

**Problem**: `.env` file is not in the correct location

**Solution**: Copy `.env` to the same folder as `TrainingQA-Helper.exe`

### "OPENAI_API_KEY not found in .env file!"

**Problem**: The `.env` file doesn't have the API key line

**Solution**: Make sure your `.env` file contains:
```
OPENAI_API_KEY=sk-proj-your-key-here
```

### File shows as ".env.txt" in File Explorer

**Solution**:

2. View â†’ Show â†’ File name extensions (check the box)
